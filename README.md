# 🎓 Student Management System

A comprehensive student management system built with Python and Streamlit.

## Features

- **Student Management**: Add, update, delete, and view student information
- **Course Management**: Create and manage courses
- **Enrollment**: Enroll students in courses
- **Grade Management**: Record and calculate grades
- **Attendance Tracking**: Mark and monitor attendance
- **Reports**: Generate academic reports and analytics
- **User Authentication**: Secure login system
